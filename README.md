# Todo List

In this project, I created a live and interactive Todo List App. It has features of adding to the list, removing from the list and editing the available list.

## Built With

- HTML
- CSS
- JavaScript

## Live Demo

[Live Demo Link](https://olamarx.github.io/Todo-List/dist)


## Getting Started

To run a local copy of this project follow these simple steps.

### Prerequisites

- Any web browser (e.g. Google Chrome)
- Any Code Editor (e.g. Vscode)

### Cloning the repository

- Create a new folder.
- Open your console in the new folder and run this command: `https://github.com/Olamarx/Todo-List.git`.
- Open index.html file with your code editor, and run it with a live server extension.
- You're done!
